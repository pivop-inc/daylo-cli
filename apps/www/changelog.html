Changelog

Last updated: July 15, 2026

Product line: Daylo v0.1 hosted API beta

A short public log for beta-facing changes. The source of truth for API shape remains docs/spec.md; this page explains what changed for users evaluating trust and access.

security status changelog contact home

§ 01July 15, 2026 - Trust Pages

  • Added static Security, Status, Changelog, and Contact pages.
  • Clarified the Daylo v0.1 shape: hosted API plus open-source CLI, API spec, and agent skill.
  • Clarified beta access: free beta, no card required, invite or waitlist.
  • Weakened homepage self-hosting language so the hosted API is the primary product path.

§ 02Current v0.1 Beta Snapshot

  • Hosted API at https://daylo.cc.
  • Public REST contract under /api/v1.
  • JSON-first CLI that runs from npm with Bun.
  • Agent skill for reading and syncing Daylo data from coding-agent workflows.
  • Withings and Tanita Health Planet support verified on the hosted production service.

§ 03Out of Scope for v0.1

  • Paid plans, credit-card collection, and billing flows.
  • Organization administration, dashboards, webhooks, and generic health coaching.
  • Medical diagnosis, treatment, weight-loss coaching, or emergency response.
